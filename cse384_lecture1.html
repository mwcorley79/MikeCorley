Course Topics (Tentative)

This is a fairly ambitious list. We cover the major (important) concepts, and delve into interesting details as time permits.

  • Linux Overview
    • History
    • Why Linux?
    • Linux test environment setup (Virtualbox and Mint):
    • Introduce shells and basic commands
  • Review Environment: (Shells, Commmands/Utilities, Coventions, etc.)
    • Foundational contsructs: shells: bash, csh, tcsh, ksh, files, permissions, groups and ownership (chmod, chown, setuid), man pages, standard (I/O) input/output streams, redirecton, pipes, vim, regular exressions grep, sed, man pages, configuration, paths, etc
    • Shell scripting (BASH), Python scripting
    • Introduction/Review C Programming Language
      • Language overview: constructs and conventions
      • C versus C++ comparison
      • compilation model
      • man pages
    • Advanced C Programming
      • pointers, type casting, memory allocation, bit masks, resource, management, ADT
      • review process (memory) layout, stacks and heap etcs.
      • abstract data type (ADT) example in C
    • LIBPCAP (network packet capture) library
      • Overview/Review TCP/IP model
      • TCPDump and Wireshark
      • C code examples for capturing and analyzing network traffic using PCAP libary
      • demonstrate pointers and ADTs with real world examples
      • packet handling/protocol decoding
